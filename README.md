# Weather Forecast App – Project Overview

The **Weather Forecast App** is a simple and responsive web application that allows users to check real-time weather details of any city.
It fetches live weather data using the **OpenWeather API** and displays important climate information in a clean user interface.

## Features

* Search weather by city name
* Displays:

  * Temperature
  * Humidity
  * Wind speed
  * Weather icon
* Responsive UI design
* Real-time API data fetching
* Error handling for invalid city names

## Technologies Used

* **HTML** → Structure of the app
* **CSS** → Styling and responsive design
* **JavaScript** → API integration and dynamic updates
* **OpenWeather API** → Live weather data source

## How It Works

1. User enters a city name.
2. JavaScript sends a request to the OpenWeather API.
3. API returns live weather information.
4. The app updates the UI dynamically with the received data.

## Main Components

### 1. Search Box

Used to input the city name and trigger weather search.

### 2. Weather Display Section

Shows:

* Temperature
* City Name
* Humidity
* Wind Speed
* Weather Condition Icon

### 3. API Integration

Uses `fetch()` method in JavaScript to retrieve weather data from the API.

## Advantages
